# Chinese Academic Expression Rules

Use this when polishing Chinese report language.

## Avoid

- Resume-like personal phrasing in research foundation sections.
- Promotional or colloquial phrases:
  - `不可逆转的趋势`
  - `疯狂吞噬`
  - `横亘的大山`
  - `井喷式爆发`
  - `完美适配`
  - `卓越`
- Repeated contrast constructions:
  - `不是……而是……`
  - `不只/不仅/不能只/不应只`

## Prefer

- `已成为重要方向`
- `持续占用`
- `核心瓶颈`
- `快速发展`
- `更好适配`
- `较好的实时计算与低能耗特性`

## Positive Rewrite Pattern

Instead of:

```text
软硬协同优化不应只停留在固定模型映射到固定硬件，而应...
```

Use:

```text
软硬协同优化需要从固定模型映射到固定硬件扩展到面向任务状态、输入变化和平台资源约束的动态协同。
```

## Term Annotation

When a specialized English term first appears, annotate if the document style requires it:

```text
大语言模型（large language model, LLM）
键值缓存（key-value cache, KV Cache）
有限元分析（finite element analysis, FEA）
```

Do not over-annotate common repeated terms after the first occurrence.
